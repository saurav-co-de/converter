# Rollout Phases

1. Foundation: monorepo, local infra, route tree, shared contracts, worker skeleton.
2. Core pipeline: real uploads, DB persistence, queue consumer, status transitions, artifact cleanup.
3. Launch tools: merge, split, compress, JPG to PDF, PDF to JPG, rotate, watermark, protect, unlock.
4. Expansion: Office conversions, organize, crop, page numbers, PDF/A.
5. Premium/business: billing, workflows, connectors, team surfaces.
6. Advanced: OCR, repair, redact, compare, sign, AI features.
